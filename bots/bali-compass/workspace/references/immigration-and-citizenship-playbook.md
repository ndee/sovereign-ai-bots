# Immigration And Citizenship Playbook

Use this file for visa, residency, permanent-stay, and citizenship orientation.

Always separate the user's goal into one of these tracks:

1. short stay or tourism
2. remote stay or longer exploration
3. local employment
4. founder or investor path
5. marriage or family-based path
6. retirement
7. permanent-stay planning
8. naturalization or citizenship research

Core rules:

- Ask for nationality first because eligibility often changes by passport
- Ask whether the user wants travel flexibility, work authorization, business control, or a path to permanence
- Treat current visa categories, duration limits, fees, and document rules as volatile
- Never recommend grey-zone workarounds as a stable plan

Citizenship and naturalization guardrails:

- Treat citizenship as a long-horizon objective, not a short-term relocation tactic
- Explain that adult dual-citizenship treatment is generally restrictive and must be verified under current law
- Do not promise that residency alone leads to citizenship
- Give the user a lawyer-ready question list instead of false certainty

Verification checklist for execution-ready users:

1. correct visa or residency category for the real activity
2. sponsor or corporate requirements
3. work-authorization implications
4. tax-residency implications
5. family and dependent treatment
6. renewal, conversion, and exit risk
7. permanent-stay or citizenship prerequisites

Recommended answer structure:

1. likely path options
2. what this path is good for
3. main red flags
4. documents and professionals needed next
5. what must be checked against current official rules
